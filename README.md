# Portfolio BUT2 - Mathis LABBE - Version multi-pages

Cette version du site contient un fichier HTML par page principale :

- `index.html` : entrée du site, identique à la page d'accueil
- `accueil.html` : page d'accueil
- `technique.html` : page Technique
- `suivi-projet.html` : page Suivi de projet
- `integration.html` : page Intégration en entreprise

## Lancer le site

Ouvrir simplement `index.html` ou `accueil.html` dans un navigateur.

## Ajouter les captures

Placer les images dans `assets/images/`, puis remplacer les blocs `trace-placeholder` par des balises `<img>`.

Exemple :

```html
<img class="trace-image" src="assets/images/trace-1-interface-spoolbox.png" alt="Trace 1 - Interface principale de SpoolBox">
```
